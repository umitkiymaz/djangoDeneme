🚀 GitHub → Ubuntu Sunucu Deploy Rehberi

  1. Terminali aç ve projenin dizinine git:
    2. cd ~/myproject
  2. Değişiklikleri ekle:
    3. git add .
  3. Commit mesajı yaz:
    4. git commit -m "Güncellemeler yapıldı"
  4. GitHub’a gönder:
    5. git push origin main

    Not: branch adın 'main' değilse ona göre değiştir.

  1. SSH ile sunucuya bağlan:
    2. ssh kullanici@sunucu_ip
  2. Proje dizinine git:
    3. cd ~/myproject
  3. Yeni kodları çek:
    4. git pull origin main

  1. Venv’i etkinleştir:
    2. source venv/bin/activate
  2. Veritabanı güncellemeleri varsa:
    3. python manage.py migrate
  3. Statik dosyaları topla:
    4. python manage.py collectstatic
  4. Uygulamayı çalıştır (örnek):
    5. python manage.py runserver 0.0.0.0:8000

    Ya da gunicorn + supervisor ile servisleri yeniden başlat.

💡 İpucu: Bu sayfayı bir yere kaydedip hızlıca açabileceğin şekilde sakla. Daha fazla otomasyon için Fabric, Ansible gibi araçları da kullanabilirsin.
